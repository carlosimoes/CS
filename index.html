Travelling Salesman Problem

The travelling salesman problem (TSP) aims to find the shortest cycle in a weighted graph that passes through all its nodes (only once). The ant colony optimization (ACO) algorithm is a possible approach for solving optimization problems on graphs like the TSP. It simulates ants that randomly go through the graph and lay down pheromone trails after finding a solution to the problem. All of this was made in Java .
